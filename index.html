AgriRental – Role-Based Equipment Booking Platform

The Agri Rental Website is a role-based web application designed to support farmers by offering a platform to rent agricultural equipment. Built using Python and Flask, the site features dedicated dashboards for Admins, Farmers, and Vendors, ensuring tailored functionality for each user type. Farmers can search equipment using K-Nearest Neighbors (KNN) logic, view availability, and make rental requests. Vendors can list machinery and manage rental status, while Admins oversee the platform. The site also integrates MySQL for database management and supports seamless payment processing, offering a complete end-to-end solution for agricultural equipment rental.

HTML CSS JavaScript Python Flask MySQL WAMP
